# 📚 Blog API

Une API REST simple pour la gestion d’articles de blog, développée avec Node.js et Express.

## 🚀 Démo en ligne
👉 https://blog-api-inf222-fs3w.onrender.com  
📖 Documentation Swagger :  
👉 https://blog-api-inf222-fs3w.onrender.com/api-docs  

## ⚙️ Fonctionnalités
- ✔️ Création d’articles  
- ✔️ Lecture de tous les articles  
- ✔️ Lecture d’un article par ID  
- ✔️ Mise à jour d’un article  
- ✔️ Suppression d’un article  

## 🛠️ Technologies utilisées
- [Node.js](htpps://nodejs.org/)  
- [Express.js](htpps://expressjs.com/)  
- [Swagger UI](https://swagger.io/)  
- UUID (identifiants uniques)
